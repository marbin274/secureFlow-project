# SecureFlow – Modelo de Dominio Open Banking (TypeScript)

## Descripción del proyecto

**SecureFlow** es un proyecto de ejemplo desarrollado en **TypeScript** que implementa un ecosistema de **Open Banking**.

---

## Estructura del proyecto

    secureflow-domain-ts/
    ├─ package.json
    ├─ tsconfig.json
    └─ src/
    ├─ index.ts # Ejecución de ejemplo del dominio
    └─ domain/
    ├─ types.ts # Tipos y enums del dominio
    ├─ CuentaBancaria.ts # Entidad Cuenta Bancaria
    └─ LibroMayor.ts # Entidad Libro Mayor


---

## Requisitos

- **Node.js** v18 o superior  
- **npm** v9 o superior  

## Instalación
    npm install
    npm run dev
